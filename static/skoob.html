Privacidade

Este site não armazena ou compartilha nenhum dado que você insere ou exporta. Todo o processamento é feito inteiramente no seu navegador.

Nenhuma informação é enviada aos nossos servidores ou a terceiros. Você pode verificar isso inspecionando o código fonte ou desconectando-se da internet após carregar esta página.

Seu ID do Skoob é utilizado apenas para acessar a API pública do Skoob para obter seus dados de leitura.

Sobre o Projeto

O Skoob Exporter é uma ferramenta que apoia o direito à liberdade de leitura e o acesso aos seus próprios dados. Acreditamos que todos devem ter controle sobre suas informações pessoais.

Esta ferramenta é código aberto e foi desenvolvida para ajudar leitores a terem mais autonomia sobre seus dados de leitura.

Não temos nenhuma afiliação oficial com o Skoob. Este projeto utiliza apenas a API pública disponibilizada pela plataforma.
